# ぷよ認識 for puyosim

「ぷよぷよ!!クエスト」のプレイ画面スクリーンショットを解析し、[puyosim.com](https://puyosim.com/) で再現可能なURLを自動生成するウェブツールです。

## 特徴

- **自動配置認識**: 画像をアップロードするだけで、盤面のぷよ配置を特定します。
- **ネクストぷよ対応**: ネクスト（控え）のぷよも認識し、シミュレーターに反映します。
- **puyosim.com 連携**: 生成されたURLをクリックするだけで、シミュレーターが立ち上がります。

## 使い方

1. `index.html` をブラウザで開きます。
2. ぷよクエのプレイ画面スクリーンショットをアップロードします。
3. 生成されたURLをクリックするか、コピーして使用してください。

## 技術スタック

- HTML5 / CSS3 / JavaScript (Vanilla)
- [OpenCV.js](https://docs.opencv.org/4.x/opencv.js) - 画像解析エンジン

## デプロイ

Netlify 等の静的サイトホスティングサービスで公開可能です。
